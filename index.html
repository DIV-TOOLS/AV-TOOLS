Cable Wiring Guides — Pinouts & Diagrams
XLR — Balanced Audio
Standard 3-pin XLR balanced connection (male to female)
PinSignalColour (typical)Notes
Pin 1Ground / ShieldBare / BlackConnect at both ends
Pin 2Hot (+)Red / WhiteNon-inverting signal
Pin 3Cold (−)Blue / BlackInverting signal
AES14 standard. Pin 2 Hot is universal. Some older Neve/Studer equipment used Pin 3 Hot — check before connecting.
XLR Balanced → RCA Unbalanced
Connecting a balanced output to an unbalanced consumer input
XLR Pin→RCANotes
Pin 2 (Hot)→Centre TipSignal conductor
Pin 1 (GND)→Outer RingGround / shield
Pin 3 (Cold)→Tie to Pin 1Ground at XLR end only
⚠ Tie Pin 3 to Pin 1 at the XLR end only — not at both ends. Expect ~6dB level drop vs balanced.
TRS Jack — Balanced / Insert
6.35mm (¼") TRS — balanced line or insert send/return
SegmentBalanced UseInsert Use
TipHot (+) signalSend (to outboard)
RingCold (−) signalReturn (from outboard)
SleeveGround / ShieldGround / Shield
Insert cables are Y-split: one TRS plug → two TS plugs (Send + Return). Do not use an insert cable for balanced line — tip only will be mono unbalanced.
Headphone TRS → Dual Balanced XLR
Splitting stereo headphone output into two balanced mono XLR channels
TRS→XLR Ch.Pin
Tip (L+)→Left XLRPin 2 (Hot)
Ring (R+)→Right XLRPin 2 (Hot)
Sleeve (GND)→Both XLRPin 1 (GND)
—→Both XLRPin 3 → Pin 1
⚠ This is unbalanced signal on a balanced connector. Tie Pin 3 to Pin 1 at the XLR end. Expect noise on long runs — use a DI box for runs over 10m.
XLR → TRS Balanced (Patch)
Standard balanced patch — XLR male to TRS jack
XLR Pin→TRS Segment

SpeakON NL4 — Speaker Wiring
Neutrik NL4 4-pole SpeakON connector pinout
PinSignalBi-amp / Mono use
1+Channel 1 PositiveFull range / HF positive
1−Channel 1 NegativeFull range / HF negative
2+Channel 2 PositiveLF positive (bi-amp)
2−Channel 2 NegativeLF negative (bi-amp)
NL2 uses only 1+ and 1−. For mono full-range use 1+ and 1− only. Never connect speaker ground to chassis/earth.
DMX512 — 5-Pin XLR
Standard DMX512 lighting control wiring (5-pin and 3-pin)
PinSignal3-Pin Equiv.
1Ground / ShieldPin 1
2Data 1 − (Cold)Pin 2
3Data 1 + (Hot)Pin 3
4Data 2 − (unused)—
5Data 2 + (unused)—
⚠ Always terminate the DMX line with a 120Ω resistor between Pin 2 and Pin 3 at the last fixture. Max 32 devices per universe without splitter. Use DMX-rated 110Ω cable — not standard mic cable.
BNC — SDI / Video / RF
75Ω BNC coaxial wiring for SDI, composite video and RF signals
ConductorSignalNotes
Centre PinVideo / RF signalDo not ground
Outer ShieldGround / ReturnConnect at both ends for video
Always use 75Ω rated cable and connectors for SDI/video. Never use 50Ω RF cable (RG-58) for video — impedance mismatch causes reflections. Terminate unused BNC outputs with 75Ω terminator.
Ethernet Cat6 — 568B Standard
TIA/EIA 568B wiring for structured cabling, Dante, AES67, HDBaseT
PinColour (568B)PairFunction
1Orange/WhitePair 2TX+
2OrangePair 2TX−
3Green/WhitePair 3RX+
4BluePair 1Unused (PoE)
5Blue/WhitePair 1Unused (PoE)

8BrownPair 4Unused (PoE)
568A swaps pairs 2 & 3 (pins 1,2 and 3,6). Use 568B throughout for new installs. For Dante/AES67 use Cat6 STP minimum. HDBaseT requires Cat6A STP for 100m at 4K.
